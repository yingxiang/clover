## zh-Hans

### 副标题

高效多窗口文件管理器

### App 描述

Clover 是一款为 macOS 打造的高效文件管理器，帮助你用更少的切换、更清晰的布局处理日常文件工作。

你可以在单窗格、双窗格、三窗口和四窗格之间快速切换，支持8种窗口布局，把不同文件夹并排打开；常用目录放在侧边栏，配合路径栏、前进后退、搜索、排序与类型筛选，查找文件更直接。

在浏览体验上，Clover 支持列表与网格两种视图、Quick Look 快速预览、拖放移动、批量选择，以及新建文件夹、文本文件和 Markdown 文件。复制、移动、重命名、删除、解压、压缩等常用操作都可以在当前工作区内连续完成，减少来回打开窗口和反复跳转目录的时间。

暂存架可以把正在整理的文件临时收在一个轻量的小架子里，先放一边，再拖到目标文件夹或其他 App 中。无论是跨目录归类、从多个位置收集素材，还是临时整理下载内容，都不必反复打开中转文件夹。

如果你正在寻找 QSpace 平替，希望保留多窗格高效管理的核心体验，又想要更简单、更小巧、更轻量的使用感受，Clover 会是一个更容易上手的选择。它专注于本地文件浏览与整理，把常用功能放在顺手的位置，适合想提升 Finder 效率、但不想被复杂功能打断节奏的用户。

### 关键词

Clover,QSpace,ForkLift,Commander One,Nimble Commander,文件管理器,双栏,多窗口,访达替代

如果你正在找 QSpace 的平替，希望保留多窗格高效率管理的核心體驗，又想要更簡單、更小巧、更輕量的使用感受，Clover 會是更容易上手的選擇。它專注於本機檔案瀏覽與整理，把常用功能放在順手的位置，適合想提升 Finder 效率、又不想被複雜功能打斷節奏的使用者。

## en

### 副标题

Efficient Multi-Window File Manager

### App 描述

Clover is a file manager for macOS built to help you handle everyday files with less switching and a clearer workspace.

Switch quickly between single-pane, dual-pane, three-window, and four-pane layouts, with 8 workspace arrangements for opening folders side by side. Keep common locations in the sidebar, then use the path bar, back and forward navigation, search, sorting, and type filtering to reach files faster.

Clover supports both list and grid views, Quick Look preview, drag and drop, multi-selection, and quick creation of folders, text files, and Markdown files. Common actions like copy, move, rename, delete, extract, and compress can be completed in the same workspace, helping you reduce window hopping and repeated folder navigation.

The Stash Shelf lets you temporarily collect files in a lightweight shelf while you organize. Set files aside, then drag them to the destination folder or another app. It is useful for sorting across folders, collecting assets from multiple locations, or clearing downloads without repeatedly opening a temporary transfer folder.

If you are looking for a lighter alternative to QSpace, Clover keeps the core value of efficient multi-pane file management while feeling simpler, smaller, and easier to pick up. It focuses on local file browsing and organization, making it a practical choice for users who want more than Finder without the weight of a more complex tool.

### App 描述

Clover est un gestionnaire de fichiers pour macOS conçu pour vous aider à traiter vos fichiers quotidiens avec moins de changements de fenêtre et une interface plus claire.

Passez rapidement d'une vue à un panneau, deux panneaux, trois fenêtres ou quatre panneaux, avec 8 dispositions d'espace de travail pour ouvrir plusieurs dossiers côte à côte. Gardez les emplacements fréquents dans la barre latérale, puis utilisez le chemin, la navigation avant et arrière, la recherche, le tri et le filtrage par type pour accéder plus vite aux fichiers.

Clover propose les vues liste et grille, l'aperçu Quick Look, le glisser-déposer, la sélection multiple, ainsi que la création rapide de dossiers, fichiers texte et fichiers Markdown. Les actions courantes comme copier, déplacer, renommer, supprimer, extraire et compresser peuvent être enchaînées dans le même espace de travail.

L'étagère temporaire permet de mettre de côté les fichiers que vous êtes en train de trier dans un espace léger, puis de les faire glisser vers le dossier de destination ou une autre app. Elle est pratique pour classer des fichiers entre plusieurs dossiers, rassembler des éléments depuis différents emplacements ou organiser les téléchargements sans ouvrir sans cesse un dossier de transit.

Si vous cherchez une alternative plus légère à QSpace, Clover conserve l'essentiel d'une gestion de fichiers multi-panneaux efficace tout en restant plus simple, plus compact et plus facile à prendre en main. C'est un bon choix pour celles et ceux qui veulent aller au-delà du Finder sans adopter un outil trop lourd.

## ja

### 副标题

高効率なマルチウインドウファイル管理

### App 描述

Clover は、macOS 向けに作られた高効率なファイルマネージャです。画面の切り替えを減らし、より見やすい作業空間で日々のファイル操作を進められます。

1 ペイン、2 ペイン、3 ウインドウ、4 ペインのレイアウトをすばやく切り替え、8 種類のワークスペース配置で複数のフォルダを並べて表示できます。よく使う場所はサイドバーにまとめ、パスバー、戻ると進む、検索、並び替え、種類フィルタで目的のファイルへすばやくアクセスできます。

一覧表示とグリッド表示、Quick Look プレビュー、ドラッグ＆ドロップ、複数選択、フォルダ・テキストファイル・Markdown ファイルの新規作成に対応。コピー、移動、名前変更、削除、展開、圧縮といった操作も同じワークスペース内で続けて行えます。

一時シェルフには、整理中のファイルを軽い小さな置き場として一時的に集められます。いったん横に置いてから、目的のフォルダや別の App へドラッグできます。複数のフォルダをまたいだ分類、いろいろな場所からの素材集め、ダウンロードの一時整理に便利です。

QSpace の代わりになる、もっと軽くてシンプルなアプリを探しているなら、Clover は扱いやすい選択肢です。マルチペイン管理の便利さはそのままに、より小さく軽快で、Finder より一歩進んだ整理体験を提供します。

## ko

### 副标题

효율적인 멀티윈도우 파일 관리자

### App 描述

Clover는 macOS를 위한 파일 관리 앱으로, 창 전환을 줄이고 더 정돈된 작업 공간에서 일상적인 파일 작업을 처리할 수 있도록 도와줍니다.

단일 창, 듀얼 패널, 3창, 4분할 레이아웃을 빠르게 전환할 수 있으며, 8가지 작업 공간 배치로 여러 폴더를 나란히 열 수 있습니다. 자주 사용하는 위치는 사이드바에 두고, 경로 막대, 앞뒤 이동, 검색, 정렬, 유형 필터를 통해 원하는 파일에 더 빠르게 접근할 수 있습니다.

목록 보기와 그리드 보기, Quick Look 미리보기, 드래그 앤 드롭, 다중 선택, 폴더·텍스트 파일·Markdown 파일 빠른 생성을 지원합니다. 복사, 이동, 이름 변경, 삭제, 압축 해제, 압축 같은 자주 쓰는 작업도 같은 작업 공간 안에서 이어서 처리할 수 있습니다.

임시 선반은 정리 중인 파일을 가볍게 모아 두는 작은 공간입니다. 파일을 잠시 올려 둔 뒤 대상 폴더나 다른 앱으로 끌어다 놓을 수 있습니다. 여러 폴더를 오가며 분류하거나, 여러 위치의 자료를 모으거나, 다운로드 파일을 임시로 정리할 때 중간 폴더를 반복해서 열 필요가 줄어듭니다.

QSpace의 가벼운 대안을 찾고 있다면 Clover는 멀티 패널 파일 관리의 핵심 장점을 유지하면서도 더 단순하고 더 작고 더 가볍게 사용할 수 있는 선택지입니다. Finder보다 효율적이지만 지나치게 복잡하지 않은 앱을 원하는 사용자에게 잘 맞습니다.
