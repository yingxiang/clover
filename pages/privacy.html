Privacy Policy

Clover Privacy Policy

Clover is designed for local file browsing and organization on macOS. This policy explains how the current version of the app handles data and system permissions.

Effective date: May 3, 2026

Clover is a macOS file manager focused on local file access. In the current version, Clover does not require user account registration and is not designed as a cloud service.

Information Clover accesses

Clover accesses files and folders that you choose to open or grant permission to on your Mac. This access is used only to display folder contents, navigate between locations, preview supported files, and perform file actions that you request, such as copy, move, rename, create, or delete.

Information Clover does not intentionally collect

  • No account credentials are required to use the current version of Clover.
  • No in-app profile, social graph, or advertising identifier is required.
  • No personal file content is intentionally transmitted to a Clover-operated server by the current version of the app.

Local storage and permissions

Clover may store workspace-related settings locally on your device, such as layout state, pane configuration, and recently granted folder access needed for normal app behavior. These settings are stored locally to make the app reopen in a familiar state and to preserve file access choices that macOS allows the app to remember.

Third-party services

Based on the current version of Clover, the app is not intended to depend on a Clover-operated account system, analytics platform, or cloud sync service for its core local file management features.

Children's privacy

Clover is not directed to children under 13, and no special child user features are provided in the current version of the app.

Changes to this policy

If Clover introduces new online features, analytics, account systems, or other data processing behavior in a future release, this privacy policy should be updated before or at the time those features become available.
