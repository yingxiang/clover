Help for Clover on macOS

Clover is built for local file browsing and organization on macOS. If something is not working as expected, the guidance below covers the most common questions and troubleshooting steps.

What Clover does

Clover is a multi-pane file manager for macOS. It supports single-pane, dual-pane, and four-pane layouts, list and grid views, Quick Look preview, drag and drop, file search, sorting, filtering, and common actions such as copy, move, rename, and delete.

Common troubleshooting

The app cannot access a folder

Make sure the folder is available on your Mac and that Clover has been granted permission when macOS requests access. If access was denied earlier, remove and reselect the folder from within the app.

A folder or file list does not refresh

Try refreshing the active pane, switching to another folder and back, or reopening the window. If the file is managed by another app or external storage, confirm it is still mounted and available.

Drag and drop or file operations fail

Check that the item is not locked, in use by another application, or located in a protected macOS directory that requires additional permission.

Preview does not appear

Quick Look support depends on the selected item type and macOS system support. Some file formats may not provide a preview.

Before requesting support

  • Confirm that your macOS version is up to date and compatible with Clover.
  • Restart Clover and try the same action again.
  • Check whether the issue affects one folder only or multiple locations.
  • Note the exact steps that reproduce the problem.

How to ask for help

When contacting support, include your macOS version, the Clover app version, and a short description of what happened. If possible, include the steps needed to reproduce the issue and whether it involves local storage, external drives, or protected folders.

If you reached this page from the App Store, please use the support contact method associated with the Clover App Store listing or developer information for the fastest response.
